Build clean, responsive pricing tables with simple shortcodes — highlight a plan, add features and a CTA.

== Description ==

Create professional, mobile-friendly pricing tables in minutes. Wrap your plans
in `[pricing]` and add a `[plan]` for each tier, with a name, price, feature
list and call-to-action. Mark one plan as featured to draw the eye.

    [pricing]
      [plan name="Starter" price="$9" period="/mo" url="#" features="1 site|Email support"]
      [plan name="Pro" price="$29" period="/mo" url="#" featured="yes"
            features="10 sites|Priority support|All add-ons"]
    [pricing]

**Free features**

* Unlimited responsive pricing tables.
* Featured-plan highlight, checkmark feature lists, CTA buttons.
* Custom accent colour.
* No page builder required.

**Pro features**

* Monthly/annual toggle.
* Ribbon labels and per-plan colours.
* Priority support.

== Installation ==

1. Install via Plugins → Add New, or upload the zip.
2. Activate.
3. Add the [pricing] / [plan] shortcodes to any page.
