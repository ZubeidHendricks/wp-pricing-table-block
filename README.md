# Pricing Table

Build clean, responsive pricing tables with simple shortcodes — highlight a
plan, add features and a CTA. Part of the
[Zub Plugin Factory](https://github.com/ZubeidHendricks/wp-plugin-factory).

## What it does

```
[pricing]
  [plan name="Starter" price="$9"  period="/mo" url="#" features="1 site|Email support"]
  [plan name="Pro"     price="$29" period="/mo" url="#" featured="yes"
        features="10 sites|Priority support|All add-ons"]
[pricing]
```

Renders responsive cards with a featured highlight, checkmark feature lists and
CTA buttons. Custom accent colour.

## Settings

**Settings → Pricing Table** — accent colour.
